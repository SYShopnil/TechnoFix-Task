## Run Locally

Clone the project

```bash
  git clone https://github.com/SYShopnil/TechnoFix-Task.git
```

Go to the project directory

```bash
  cd
  TechnoFix-Task
```

Install dependencies

```bash
  npm install || npm i
```

Start the server

```bash
  npm run dev
```

Start the StoryBook

```bash
  npm run storybook
```
