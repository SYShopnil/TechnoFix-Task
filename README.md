## Run Locally

Clone the project

```bash
  git clone https://github.com/SYShopnil/AgencyTaskLH.git
```

Go to the project directory

```bash
  cd
  AgencyTaskLH
```

Install dependencies

```bash
  npm install || npm i
```

Start the server

```bash
  npm run dev
```

Start the StoryBook

```bash
  npm run storybook
```
